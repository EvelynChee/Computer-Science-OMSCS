This folder contains all files that were generated using MATLAB and 
ABAGAIL used in the analysis of the four algorithms as part of 
Assignment 2.

In order to recreate these results, please run the MATLAB code from 
the following folders;

Optimization Problems:
Rastrigin's
$ASSIGNMENT_HOME/code/MATLAB/solveRas.m
TSP:
$ASSIGNMENT_HOME/code/MATLAB/solveTSP.m
One-Max
$ASSIGNMENT_HOME/code/ABAGAIL/src/opt/test/CountOnesTest.java

Neural Network:
$ASSIGNMENT_HOME/code/MATLAB/solveNN.m

Images folder contains all the results generated for the assingment. 
This can be loaded in MATLAB.
